Subpackages and Descriptions
----------------------------

There are several subpackages in InflGame that make modeling an influence game a simpler process. Therese are in listed in the following:

==================  =============================================
Subpackage          Description
==================  =============================================
`adaptive`          Adaptive (Gradient ascent) dynamics
`MARL`              Multi-agent learning
`domains`           Domains from the paper
`kernels`           Predefined influence kernels from the paper
`utils`             Utility functions for across the package
==================  =============================================
