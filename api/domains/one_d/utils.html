Utilities#

1D Utility Module#

This module provides utility functions for setting up and managing 1D domains in influencer games. It includes tools for creating color schemes, plotting critical values, and handling domain-specific configurations.

The module is designed to work with the InflGame package and supports creating structured 1D environments for simulations involving agent dynamics and resource distributions.

Dependencies:#

  • InflGame.utils

Usage:#

The critical_values_plot function can be used to plot critical values for a given resource distribution and number of agents, while the color_list_maker function generates random colors for agents.

Plot critical values given a resource distribution and number of agents t_* (assume symmetric splitting).

Parameters:

  • num_agents (int) – Number of agents.

  • bin_points (np.ndarray) – Points representing bins for resources.

  • resource_distribution (torch.Tensor) – Distribution of resources.

  • axis (plt.Axes) – Matplotlib axis to plot on.

  • reach_start (float, optional) – Starting reach value, defaults to 0.3.

  • reach_end (float, optional) – Ending reach value, defaults to 0.

  • refinements (int, optional) – Number of refinements for splitting, defaults to 2.

  • crit_cs (str, optional) – Color scheme for the plot, defaults to ‘Greys’.

Returns:
